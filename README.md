# 📊 YouTube Data Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on a YouTube dataset using Python. The analysis focuses on data exploration, preprocessing, statistical analysis, and identifying key insights from video and channel-level information.

## Key Features
- Data loading and exploration
- Dataset structure and summary statistics
- Missing value analysis
- Data cleaning and preprocessing
- Date-time transformation and feature extraction
- Monthly video publishing trend analysis
- Channel-wise and video-wise analysis
- Top and bottom performing videos based on likes and dislikes
- Views vs. dislikes analysis
- January-specific video analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Skills Demonstrated
- Exploratory Data Analysis (EDA)
- Data Cleaning & Preprocessing
- Feature Engineering
- Statistical Analysis
- Data Aggregation & Filtering
- Insight Generation

## Outcome
The project demonstrates how Python can be used to clean, analyze, and interpret YouTube video data to uncover trends in content publishing, user engagement, and channel performance.
